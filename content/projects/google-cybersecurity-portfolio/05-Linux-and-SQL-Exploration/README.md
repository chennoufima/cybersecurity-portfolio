# Linux and SQL for Cybersecurity

## Overview
This folder contains practical applications of Linux administration and SQL querying for security purposes. I practiced using the Linux terminal to manage file permissions and used SQL to filter through large organizational databases to identify security events.

## Technical Skills
- **Linux:** Managing users/groups, modifying file permissions (`chmod`), and navigating the hidden file system.
- **SQL:** Writing queries to extract specific data from relational databases using filters (`WHERE`, `AND`, `OR`) and joins.
- **Security Application:** Identifying unauthorized login attempts and ensuring the Principle of Least Privilege (PoLP).

## Projects & Artifacts
1. **[Linux Permissions Log](./linux-permissions.md)**: A record of securing system files using the CLI.
2. **[SQL Security Queries](./sql-queries.md)**: A collection of queries used to investigate security logs.
