# Foundations of Cybersecurity

## Overview
This folder contains artifacts related to the fundamental concepts of cybersecurity. It explores the core mission of a security professional: protecting the Confidentiality, Integrity, and Availability (CIA) of data, and learning from historical security events to better defend modern networks.

## Learning Objectives
- Define the role of a cybersecurity analyst in an organization.
- Apply the **CIA Triad** to real-world business scenarios.
- Analyze historical security incidents to identify patterns in threat actor behavior.
- Understand the 8 CISSP security domains.

## Projects & Artifacts
1. **[CIA Triad Analysis](./cia-triad-summary.md)**: A breakdown of how organizations balance security priorities.
2. **[Historical Attack Case Study](./morris-worm-analysis.md)**: An analysis of the Morris Worm and its impact on cybersecurity history.
